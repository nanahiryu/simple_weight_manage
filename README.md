## 作った目的
* firebase auth, store, emulater辺りをセットアップから触りたい
* Next app router使いたい
* chart.js使ってみたい

## コマンド
* `npm run dev`: localhostを立ち上げる
* `npm run firebase`: firebase emulaterを立ち上げる
* `npm run firebase:export`: dbの差分を`data/v0`にexportする
